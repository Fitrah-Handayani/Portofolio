Penjelasan dari Sintaks Sign Up & Login

<form>: Elemen ini digunakan untuk membuat form di HTML. Dalam kasus ini, ada dua form: satu untuk Sign Up dan satu lagi untuk Login. <form> <label for="Email">Email:</label> <input type="text" id="Email" name="Email" required> <label for="newUsername">Username:</label> <input type="text" id="newUsername" name="newUsername" required> <label for="newPassword">Password:</label> <input type="password" id="newPassword" name="newPassword" required> <label for="Password">Konfirmasi Password:</label> <input type="password" id="Password" name="Password" required> <input type="submit" value="Sign Up"> </form>

<label>: Digunakan untuk memberi label pada input. <label for="newUsername">Username:</label>

<input>: Ini adalah elemen input di form, yang memungkinkan pengguna memasukkan informasi. Jenisnya bisa teks, password, atau email. <input type="text" id="newUsername" name="newUsername" required>

<input type="submit">: Tombol submit digunakan untuk mengirim form. <input type="submit" value="Sign Up">

Penjelasan syntax js agar bisa berpindah page

function toggleForm(formType): Fungsi ini digunakan untuk mengubah tampilan antara form Login dan Sign Up. function toggleForm(formType) { ... }

if (formType === 'login'): Kondisi ini memeriksa apakah tipe form yang akan ditampilkan adalah Login. Jika ya, maka form Sign Up akan disembunyikan, dan form Login akan ditampilkan. if (formType === 'login') { ... }

event.preventDefault(): Digunakan untuk mencegah perilaku default dari form, yaitu mencegah halaman melakukan refresh saat form disubmit. event.preventDefault();

localStorage.setItem(): Fungsi ini digunakan untuk menyimpan data di browser. Dalam konteks ini, username dan password yang dimasukkan oleh pengguna disimpan di localStorage. localStorage.setItem('username', newUsername);

document.querySelector(): Digunakan untuk memilih elemen HTML berdasarkan selector yang diberikan. Pada kasus ini, elemen form Sign Up dan Login dipilih. document.querySelector('#signupForm form')

alert(): Fungsi ini digunakan untuk menampilkan pesan dialog kepada pengguna. Misalnya, untuk memberitahukan bahwa registrasi berhasil atau login gagal. alert('Registrasi berhasil! Silakan login.');

window.location.href: Digunakan untuk mengalihkan pengguna ke halaman lain setelah berhasil login. Dalam hal ini, pengguna akan dialihkan ke halaman index.html. window.location.href = "index.html";

var storedUsername = localStorage.getItem('username'): Kode ini mengambil data username yang telah disimpan di localStorage untuk memvalidasi login. var storedUsername = localStorage.getItem('username');
